Nodeâ€‹

  1. Install nvm
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
  1. Instal NodeJS 14
$ nvm install 14
  1. Check installation
$ nvm -v
$ node -v
$ npm -v

Pythonâ€‹

  1. Go to this link and download the installer and then install.
  2. Install Miniconda
  3. Create virtual environment
# To init conda
$ /home/chiao/miniconda3/bin/conda init zsh

# Then you can use conda as a short command
$ conda create -n submarine-dev python=3.6
  1. Activate environment
$ conda activate submarine-dev
  1. Decativate current environment
$ conda deactivate
