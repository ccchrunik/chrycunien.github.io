Java

SDK#

  1. Install sdkman
$ curl -s "https://get.sdkman.io" | bash
  1. Restart or source the ~/.zshrc or ~/.bash_profile
  2. Find a version to install
$ sdk list java
  1. Install JDK 8
$ sdk install java 8.0.292.j9-adpt

IntelliJ#

  1. Download the tar.gz file
  2. Extract directory to /opt
$ sudo tar -xzf ideaIU.tar.gz -C /opt
  1. Rename directory
$ cd /opt$ sudo mv ideaxxxxx idea-IU
  1. Add the script in PATH env variable (I use zsh)
export PATH="${PATH}:/opt/idea-IU/bin"
  1. Run the script (you have to login or enter license at the first time)
$ idea.sh

Maven#

  1. Download zip or tar.gz (maven 3.6.3)
  2. Extract the file (I use unzip or you can follow this link)
$ unzip apache-maven-3.6.3-bin.zip
  1. Move the directory to /opt 
$ sudo mv apache-maven-3.6.3 /opt/maven
  1. Add the bin location to PATH variable
export PATH="${PATH}:/opt/maven/bin"
  1. Check success
$ mvn -v
