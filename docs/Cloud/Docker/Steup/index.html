Setup

I'm using Mac.

Install Docker Desktop#

One way to use docker is to install docker desktop. You can find it in the official webpage of Docker.

Install via Minikube#

Another way is to setup a minikube cluster with the help of hyperkit vm.

  • Install hyperkit
$ brew install hyperkit
  • Install docker cli
$ brew install docker
  • Install Minikube and kubectl
$ brew install minikube kubectl
  • My simple setup script
#!/bin/bash
minikube ip &> /dev/null
# Make sure no minikube cluster is runningif [[ "${?}" -ne 0  ]]then    echo "Minikube is down!"else    echo "Minikube is up!"    docker_setup    exit 1fi
echoecho "Starting Minikube..."echo
echo "No. of Parameters: ${#}"
CPUS="${1}"MEMORY="${2}"g
if [[ "${#}" -eq 0  ]]then    echo "Using default setting!"elif [[ "${#}" -eq 1 ]]then    echo "Setting cpus as: " "$CPUS"    minikube config set cpus "$CPUS"else    echo "Setting cpus as: " "$CPUS"    echo "Setting memory as:" "$MEMORY"    minikube config set cpus "$CPUS"    minikube config set memory "$MEMORY"fi
# Clean up all former resourcesechoecho "Deleting all resources..."minikube delete# Start minikubeechominikube start
  • Point docker-cli to docker daemon in minikube (You have to do it when you open a new terminal session)
$ eval $(minikube docker-env)
