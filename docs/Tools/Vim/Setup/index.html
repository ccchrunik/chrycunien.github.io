Vim Setup

Settings#

  • Go to this website and download the font monokai.
  • Put the following configuration in ~/.vimrc
  • Simplified Version
set nocompatible
"#######################################################syntax enableset numberset norulerset ignorecaseset smartcaseset incsearchset aiset siset expandtabset tabstop=4set softtabstop=4set shiftwidth=4set smarttabset confirmset backspace=indent,eol,startset history=500set showcmdset showmodeset nowrapset autowriteset mouse=aset fileencodings=utf-8,ucs-bom,gb18030,gbk,gb2312,cp936set termencoding=utf-8set encoding=utf-8set wildmenuset lbr"""#######################################################" Colorset t_Co=256" colo torteset cursorline" set cursorcolumnset hlsearch"set background=darkset scrolloff=5
"#######################################################" statuslineset laststatus=2
