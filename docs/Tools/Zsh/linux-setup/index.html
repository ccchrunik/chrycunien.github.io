Ubuntu Setup

Zsh Setup#

  1. Type the following commands, and then restart.
$ sudo apt install zsh$ chsh -s $(which zsh)
  1. Then follow a series of instructions to install zsh.

Zim Setup#

  1. Install zim
$ $ curl -fsSL https://raw.githubusercontent.com/zimfw/install/master/install.zsh | zsh
  1. Run the following commands
$ vim ~/.zimrc
# Then add the following line to bottom of ~/.zimrc# `zmodule romkatv/powerlevel10k`
$ zimfw install
  1. Fully restart the terminal.
